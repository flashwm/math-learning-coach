# 几何图规范

## 出图前

- 明确所有关键点、线段、角度、相等关系、垂直关系、共线关系。
- 折叠问题要验证折痕是否为对应点连线的垂直平分线。
- 中点、交点、垂足不能默认重合，必须逐一确认。
- 坐标图和函数图必须用同一套坐标到屏幕的转换规则。

## 出图时

- 优先使用确定性的 SVG 或坐标绘图。
- 标签要避开点、线、箭头和公式。
- 每一步只高亮当前使用的关键关系。
- 如果源图有固定朝向，主图要保持和源图一致。

## 出图后

- 检查点的位置、线段关系、垂直关系、共线关系是否正确。
- 检查文字是否重叠、遮挡或跑出画布。
- 检查学生能否从图中看出当前步骤的关键关系。
