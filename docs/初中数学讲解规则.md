# 初中数学讲解规则

## 默认原则

- 默认使用中文讲解。
- 先给结论，再讲思路，最后展开计算。
- 讲思路时先说明“为什么这样想”，不要直接堆公式。
- 用户要求初中范围时，严格避免高中或大学表述。

## 禁用或慎用表达

初中讲解中默认不要使用：

- 向量、方向向量
- 点积、叉积
- 投影公式
- 高等几何或解析几何的快捷结论

## 推荐替代表达

- 用“两点的横向差、纵向差”解释坐标变化。
- 用“斜率相等，所以三点共线”解释共线。
- 用“两条直线斜率乘积为 -1”或“构造直角三角形”解释垂直。
- 用“中点坐标公式”“垂直平分线”“相似”“勾股定理”替代高级表达。
