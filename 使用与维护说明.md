# math-learning-coach 插件使用与维护说明

这份说明用于快速了解：数学学习讲解插件能干什么、平时怎么使用、发现问题后怎么改进、后期怎么查看维护记录。

## 一句话说明

`math-learning-coach` 是一个面向数学学习的 Codex 个人插件。它的重点不是只给答案，而是把题目讲成学生能理解的学习材料，尤其适合拍照题、初中数学、几何题、函数题和网页讲义。

## 它能干什么

### 1. 讲数学题

适合：

- 拍照上传一道数学题，让它识别题目并讲解。
- 先给结论，再讲思路，再列计算。
- 限制在初中、人教版、课堂可接受的方法内。
- 避免向量、点积、投影等超纲说法。

常用说法：

```text
用数学学习讲解讲这道题，先讲思路再计算。
```

### 2. 出网页讲义

适合：

- 把一道题整理成网页讲义。
- 每一步配图说明。
- 结论、思路、计算、易错点分区展示。
- 几何图、函数图、信息图要求清楚、不重叠、不乱标。

常用说法：

```text
把这道题做成网页讲义，每一步都配图，适合初中学生复习。
```

### 3. 沉淀长期规则

如果你希望某个反馈以后一直生效，可以说：

```text
这个规则以后都要这样，改进数学学习讲解插件，并同步到数技项目。
```

插件应先修正当前题，再把可复用规则同步到项目和日志。

## 重要文件

- `plugin-source\math-learning-coach`：唯一正式插件源码。
- `plugin-source\math-learning-coach\skills\math-learning-coach\SKILL.md`：数学讲解规则。
- `docs\`：长期规范，例如初中语言边界、几何图规范、讲义规范、错题归因模板。
- `examples\`：典型题型测试集。
- `logs\skill-improvements.md`：长期连续记录每次技能改进。
- `CHANGELOG.md`：版本级摘要。
- `IDEAS.md`：以后可能要做的想法。

## 常用维护命令

从数技项目发布到 Codex 个人插件目录：

```powershell
cd E:\CodeX项\数技
.\scripts\publish-plugin.ps1
```

安装或刷新个人插件：

```powershell
codex plugin add math-learning-coach@personal
```

查看插件状态：

```powershell
codex plugin list
```

## 每次长期改进后要检查

- 当前题是否已经按新规则重新讲好。
- `SKILL.md` 是否只写了通用规则，没有写一题专属细节。
- 相关 `docs` 或 `examples` 是否需要补充。
- `logs\skill-improvements.md` 是否记录了来源、原因、改动文件和同步状态。
- `CHANGELOG.md` 是否补了摘要。
- 插件是否已发布并刷新。
- `codex plugin list` 中 `math-learning-coach@personal` 是否为 `installed, enabled`。
- 旧独立技能目录 `C:\Users\xiao\.codex\skills\math-learning-coach` 是否不存在。

## 推荐使用习惯

平时学题时不用想着维护项目。只需要区分两句话：

```text
这个只是当前题调整。
```

或者：

```text
这个以后都要这样，改进数学学习讲解插件并同步到数技项目。
```

这样可以把“正在学习”和“长期改进插件”分开，不打断学习，也不会让好经验散在不同会话里。
